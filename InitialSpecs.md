# Project overview

A simple Visual Studio extension compatable with Visual Studio 2022 and 2026. 

**NOTE**: Not VSCode!

## Functionality

- A user should be able to right click in any text editor window and see a new entry in the context menu labelled `Copy path and line`.
- On clicking, the path to the file and the line number selected are added to the clipboard

## Layout
`{path} (Line: {lineNumber})`
eg. "C:/Test.cs (Line: 200)"
